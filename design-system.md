# TanHaoMao Design System

Version 1.0

---

# Language

Specification Language

English

Product Language

Simplified Chinese

All interface copy must be written in Chinese.

Never generate English UI.

---

# Purpose

Build a calm, warm and trustworthy mobile experience.

The interface should feel like a real consumer product instead of a portfolio concept.

Visual design should never become the focus.

Users should focus on the cats.

# Design Principles

## Experience Before Decoration

Every visual element must serve usability.

Avoid decorative graphics without purpose.

---

## Less Is More

Prefer removing one component

instead of adding one more.

---

## Calm Interface

The interface should reduce pressure.

Avoid visual competition.

---

## Soft Personality

Rounded corners.

Gentle colors.

Comfortable spacing.

No sharp feeling.
